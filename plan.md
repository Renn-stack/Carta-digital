# Plan: Página Web Romántica por Capítulos para Dorian

## Objetivo
Crear una página web estilo "tech dark" que presente una carta de amor estructurada por capítulos, con animaciones fluidas y una transición de colores desde negro hasta rosa/rojo al final.

## Estructura de Capítulos (basada en la carta)
1. **Capítulo I - La Conexión**: "¿Qué significa realmente estar unidos a alguien?"
2. **Capítulo II - Confianza**: La intimidad y el primer momento juntos
3. **Capítulo III - Sin Etiquetas**: Uninamus / Unamus - más allá de las palabras
4. **Capítulo IV - Libre**: Amor sin competencia ni comparación
5. **Capítulo V - Admiración**: Lo que admira de él
6. **Capítulo VI - Caballerosidad**: Detalles que enamoran (cubo misterioso, página web, manualidades)
7. **Capítulo VII - Naturaleza**: Sobre las flores
8. **Capítulo VIII - Descanso**: La mente descansa con él
9. **Capítulo IX - Resiliencia**: Cita de Katy Perry + lluvia y arcoíris
10. **Capítulo X - Crecimiento**: Transformación personal
11. **Capítulo XI - Color**: Conclusión con transición a rosa/rojo

## Diseño Visual
- Fondo: Negro profundo (#0a0a0a a #000000)
- Texto: Blanco/gris claro inicial, transición a rosa (#ff6b9d) y rojo (#e84545) al final
- Tipografía: Monospace/tech para títulos de capítulos, serif legible para el contenido
- Efectos: Partículas sutiles, gradientes animados, transiciones suaves entre capítulos
- Animaciones: Scroll-reveal, typewriter effects, glow effects en los últimos capítulos

## Tecnología
- React + TypeScript + Tailwind CSS
- Framer Motion para animaciones
- Canvas/Partículas para efectos visuales
